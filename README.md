# CampusLearn

Full-stack e-learning app for SEN371 Final Project.

## Stack
- Backend: Node.js, Express, Prisma, PostgreSQL, JWT
- Frontend: React (Vite), Tailwind CSS

## Run locally
\\\ash
# Backend
cd backend
npm install
npx prisma migrate dev
node scripts/seed.js
npm run dev

# Frontend (new terminal)
cd ../frontend
npm install
npm run dev
\\\
